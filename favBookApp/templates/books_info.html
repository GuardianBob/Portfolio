Objectives:
  • Allow users to add books with reviews.
  • Allow users to delete their reviews.
  • Display latest three reviews on the front page.
Additional:
  • Utilize Django forms for input validation and standardization
  • Display review ratings in stars with django templating engine.
  • Verify Authors don't exist in database prior to adding them.

      This was an optional assignment used as a practice for the final full-stack Python exam. The goal was to be able to finish building this web-app within a specific time-period in order to pass: higher grades required shorter times for completion.
      The user should be able to log in utilizing the login application where they would be taken to the home page which displays the three most recent book reviews along with a list of all books that have been added to the database.
      Users are able to add books, add reviews to other books, delete their own reviews, and even add books to a list of their favorites.
      The web-app utilizes multiple database tables linked together by one-to-many and many-to-many relationships.
